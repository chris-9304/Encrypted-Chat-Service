# Security Policy

## Supported Versions
EncryptiV is in early preview (0.x). Security patches will be applied to the master branch. 

## Reporting a Vulnerability
Security contact: SECURITY-CONTACT-TBD (Update before v1.0).

## Out-of-Scope (Phase 1)
As detailed in [THREAT_MODEL.md](THREAT_MODEL.md), the following are explicitly out-of-scope for the Phase 1 LAN messenger:
- Cross-network messaging security
- Traffic analysis resistance
- Server-relayed offline delivery threats
- Protection against physical access to unlocked endpoint data
- Anonymity from the local LAN administrator
